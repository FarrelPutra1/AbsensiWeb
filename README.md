# Portal Absensi Web - Frontend Web

Website ini merupakan portal manajemen absensi sekolah yang digunakan oleh admin atau guru untuk memantau data kehadiran siswa secara realtime. Tampilan web ini sudah dideploy dan bisa diakses langsung melalui Vercel.

## Teknologi Utama
- HTML / CSS / JavaScript 
- Vercel Deployment

## Fitur Utama Web
- Halaman dashboard monitoring kehadiran siswa.
- Integrasi data realtime dengan server API di Railway.
- Tampilan riwayat absensi harian.
